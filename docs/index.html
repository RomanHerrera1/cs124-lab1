Creating a New Item

An empty list

This shows an empty list. Now we want to press the button "Create new item +" to add something to our list. Let us add "Buy new John Grisham book" to our list.

After pressing the button "Create new item +"

After pressing the button, a text input box shows up that we enter our task into. Let us add "Buy new John Grisham book" into the text input box. Once we are finished typing out the task we want to add, we press the "Add" button.

After pressing the button "Add"

After pressing the button, the text will be added to the list. In our case, we pressed the "Add" button, and now we get "Buy new John Grisham book" as an activity on our list that we can check off when we are done with it.

Adding an Additional New Item

A non-empty list

This shows a non-empty list, and in our case, we have "Buy new John Grisham book" as a list item already. Now we want to add another item, say "Eat Lunch". First, we press the button "Create new item +" to add another task to our list.

After pressing the button "Create new item +"

After pressing the "Create new item +" button, a text input box shows up under the rest of the list. We type our task into that box. Let us add "Eat Lunch" into the text input box. Once we are finished typing out the task we want to add, we press the "Add" button.

After pressing the button "Add"

After pressing the "Add" button, the text will be added to the list as another checkbox. Notice that the list maintains the order in which the tasks were added. In our case, we pressed the "Add" button, and now we get "Eat Lunch" as an activity on our list that we can check off when we are done with it.

Marking an Item As Completed

A non-empty list

This shows a non-empty list, and in our case, we have "Call Mom" as a list item already. Now we want to mark the item as complete. Simply press the checkbox or the list item name to mark it as done.

After pressing the checkbox or task

After pressing the pressing the checkbox or the text itself, the task is complete. This is visually represented by the checked checkbox but also the strikethrough of the task so you know to move onto other tasks.

Renaming an Existing Item

A non-empty list

This shows a non-empty list, and in our case, we have "Text John" and "Eat Lunch" as list items already. Now we want to rename "Text John" to "Text John about the bank statements. First, we press the button "Rename".

After pressing the button "Rename"

After pressing the "Rename" button, text input boxes reappear for each entry of the list. Note that the text boxes will start filled in already with the tasks prior to renaming. This gives the option to rename any of the items at this point. To rename the "Text John" task, we click into the text box that contains "Text John" and type our new task name into that box. Let us change the text to "Text John about bank statements" in the text input box. Once we are finished renaming all the items, we press the "Done" button that exists where the "Rename" button was prior to pressing.

After pressing the button "Done"

After pressing the "Done" button, the text boxes will disappear. All unchanged task items will appear as they did before. The items we renamed however reflect the changes. The items will all be checkboxes again, and will maintain the order they were in before. Also note that the "Rename" button reappears in its previous spot.

Showing Only Uncompleted Items

A non-empty list

This shows a non-empty list, and in our case, we have "Text John" and "Eat Lunch" as list items already. In this case, since "Text John" is checked off and has strikethrough, we know that it is completed. Let's say we want to show only uncompleted tasks so we can better focus on what we need to do. First, press the "Show Uncompleted" button.

After pressing the button "Show Uncompleted"

After pressing the "Show Uncompleted" button, all items that were checked off will disappear. Note that the space the completed list items had is now taken by uncompleted tasks and that the order of uncompleted tasks is maintained from prior. Also, to provide a way to return to seeing all tasks, the "Show Uncompleted" button is now a "Show All" button. If pressed, all completed tasks will become visible again.

Deleting All Completed Tasks

A non-empty list

This shows a non-empty list, and in our case, we have "Text John" and "Eat Lunch" as list items already. Now we want to remove "Text John" since it has been completed. First, we press the button "Delete All Completed".

After pressing the button "Delete All Completed"

After pressing the "Delete All Completed" button, nothing will happen except that the "Delete All Completed" is now a "Confirm Delete" button. This is to help prevent accidental deletions from a mis-click. If you are sure you want to delete the completed tasks, click the "Confirm Delete" button.

After pressing the button "Confirm Delete"

After pressing the "Confirm Delete" button, all checked off tasks will be deleted. The space the completed list items had is now taken by uncompleted tasks and that the order of uncompleted tasks is maintained from prior. Note that this is not undoable.
